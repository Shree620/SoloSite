Here are some web resources that I found useful to build this site:

  • MDN Web Docs

    Comprehensive resource for web developers, providing detailed documentation on HTML, CSS, JavaScript, and more.

  • W3Schools

    Offers tutorials and references on web development languages such as HTML, CSS, JavaScript, PHP, SQL, and more.

  • freeCodeCamp

    Provides interactive coding lessons, articles, and a large community of developers.

  • Codecademy

    Interactive platform offering coding lessons in various languages including HTML, CSS, JavaScript, and more.

  • Udemy

    Marketplace for online courses, including a vast array of web development courses.

  • Coursera

    Offers courses from universities and colleges around the world, including web development and computer science.

  • edX

    Provides courses from leading institutions on web development and other tech-related topics.

  • Pluralsight

    Offers technology and creative courses, including web development, software development, and IT ops.

  • Stack Overflow

    Community of developers answering questions on a wide range of programming topics.

  • Reddit Web Dev

    Subreddit for web developers to share resources, discuss topics, and ask questions.

  • DEV Community

    Platform where developers share articles, tutorials, and insights about web development.

  • Dribbble

    Community for designers to share screenshots of their work, including web design inspiration.

  • Behance

    Showcase and discover creative work, including web design projects.

  • Awwwards

    Showcases the best web design and development from around the world, offering inspiration and insights.

  • CSS-Tricks

    Tips, tricks, and tutorials on CSS and web design.

  • Bootstrap

    Popular framework for building responsive, mobile-first websites.

  • React

    JavaScript library for building user interfaces.

  • Vue.js

    Progressive framework for building user interfaces.

  • Angular

    Platform for building mobile and desktop web applications.

  • Tailwind CSS

    Utility-first CSS framework for rapidly building custom user interfaces.

  • Chrome DevTools

    Set of web developer tools built directly into the Google Chrome browser.

  • Firefox Developer Tools

    Set of tools built into Firefox for web development.

  • jQuery

    Fast, small, and feature-rich JavaScript library.

  • Font Awesome

    Icon toolkit based on CSS and LESS.

  • GitHub

    Platform for version control and collaboration, allowing developers to host and review code, manage projects, and build software together.

  • GitLab

    Web-based DevOps lifecycle tool that provides a Git repository manager.
